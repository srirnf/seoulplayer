# 별마당 도서관(책 찾기) — Unity 씬 셋업 가이드

`02_Scripts`의 스크립트들을 씬에 연결하는 방법입니다. 순서대로 따라하면 게임이 돌아갑니다.

## 0. 준비
- Unity Hub에서 `gameproject` 열기 (스크립트가 자동 컴파일됨)
- `Edit > Project Settings > Editor` → Asset Serialization = **Force Text**, Version Control = **Visible Meta Files**

## 1. 책 데이터(BookData) 만들기 — 42권
1. Project 창에서 `Assets/09_Data` 폴더로 이동
2. 우클릭 → `Create > 별마당도서관 > Book Data`
3. 만들어진 에셋 선택 후 인스펙터에서 채우기:
   - **id**: 고유값 (book_01 …)
   - **title**: 책 제목
   - **spineSprite**: 책등 이미지(스프라이트) 드래그
   - **features**: 목표 안내 문구 (예: "빨간색, 두꺼운 책, 금장식")
4. 42권 반복 (복사-붙여넣기 후 값만 수정하면 빠름)

> 💡 처음엔 5~6권만 만들어 테스트하고, 동작 확인 후 42권으로 늘리세요.

## 2. 플레이어
1. 빈 GameObject 생성 → 이름 `Player`
2. 컴포넌트 추가:
   - `Sprite Renderer` (플레이어 탑뷰 이미지)
   - `Rigidbody2D` → **Gravity Scale = 0**, Constraints에서 **Freeze Rotation Z** 체크
   - `Collider2D` (예: Capsule/Circle) — **몸통 충돌용**, Is Trigger ❌
   - 또 하나의 `Collider2D` (Circle, 크게) — **상호작용 범위용**, Is Trigger ✅
   - `PlayerController` 스크립트
   - `PlayerInteraction` 스크립트
   - `AudioSource` (걷기 소리, Loop 체크, Play On Awake ❌) → PlayerController의 `Footstep Source`에 연결
3. PlayerController의 Move Speed 조절(기본 4)

## 3. 책장(Bookshelf) 배치
1. 책장 오브젝트 생성 (Sprite Renderer = 책장 탑뷰 이미지)
2. `Collider2D` 추가 → **Is Trigger ✅** (상호작용 범위)
3. `Bookshelf` 스크립트 추가
4. `books` 리스트에 이 책장에 꽂을 BookData 에셋들을 드래그
5. 책장을 여러 개 복제해 도서관 맵 구성 (각 책장마다 books 다르게)

## 4. 매니저
1. 빈 GameObject `GameManager` 생성 → `GameManager` 스크립트 추가
   - Time Limit(제한시간), Target Count(찾을 책 수) 설정
2. 빈 GameObject `AudioManager` 생성 → `AudioManager` 스크립트 추가
   - BGM Source / SFX Source 용 `AudioSource` 2개 연결
   - 효과음 클립(클릭/정답/오답) 연결, BGM Source의 clip에 BGM 연결

## 5. UI (Canvas)
`GameObject > UI > Canvas` 생성 후 그 안에 구성:

| UI 요소 | 종류 | 연결할 곳 (GameUIManager / BookshelfView) |
|---------|------|-------------------------------------------|
| 시간 텍스트 | TMP Text | GameUIManager.timerText |
| 찾은 수 텍스트 | TMP Text | GameUIManager.foundText |
| 목표 책등 이미지 | Image | GameUIManager.targetThumbnail |
| 목표 특징 텍스트 | TMP Text | GameUIManager.targetFeatureText |
| F 안내 | GameObject | GameUIManager.interactPrompt |
| "맞습니다" 팝업 | GameObject | GameUIManager.correctPopup |
| "아닙니다" 팝업 | GameObject | GameUIManager.wrongPopup |
| 승리 패널 | GameObject | GameUIManager.winPanel |
| 패배 패널 | GameObject | GameUIManager.losePanel |

1. Canvas에 빈 오브젝트 `UIManager` 추가 → `GameUIManager` 스크립트 → 위 표대로 연결
2. **책장 정면뷰 패널** 만들기:
   - Canvas 안에 Panel 하나 (`ShelfViewPanel`) — 평소엔 꺼둠
   - 안에 책장 정면뷰 배경 Image + 책 버튼들이 들어갈 빈 오브젝트(`BookSlots`, **Grid Layout Group** 추가)
   - Canvas에 빈 오브젝트 `BookshelfView` → `BookshelfView` 스크립트:
     - panel = ShelfViewPanel
     - bookSlotParent = BookSlots
     - bookButtonPrefab = (아래 6번에서 만든 프리팹)
     - closeButton = 닫기 버튼(선택)

## 6. 책 버튼 프리팹
1. Button(UI) 생성 → 안에 책등용 Image 배치
2. `BookButton` 스크립트 추가 → spineImage에 그 Image 연결
3. `03_Prefabs`로 드래그해 프리팹화 → 씬의 버튼은 삭제
4. 이 프리팹을 BookshelfView의 bookButtonPrefab에 연결

## 7. 플레이 테스트
- ▶ 재생 → WASD 이동, 책장 근처에서 **F** → 정면뷰 → 책 클릭
- 안내 UI의 특징과 맞는 책을 고르면 정답 처리되는지 확인

## 자주 나는 문제
- **안 움직임** → Rigidbody2D Gravity Scale 0인지, 상호작용 트리거 콜라이더와 몸통 콜라이더를 헷갈리지 않았는지 확인
- **F 눌러도 안 열림** → 책장 Collider의 Is Trigger ✅, 플레이어에 Is Trigger 콜라이더 있는지 확인
- **소리 안 남** → AudioSource 연결 및 클립 할당 확인
- **TMP 관련 창** 뜨면 "Import TMP Essentials" 클릭
