# 별마당 도서관 2.5D 셋업 가이드

3D 도서관을 WASD로 걷다가(45도 뷰), 책장에 다가가 **E**를 누르면 2D 책 찾기 화면이 뜨는 구성.
스크립트는 `Games/Library/Scripts/` 에 있고, 아래는 **Unity에서 직접 하는 조립**입니다.

## 0. 에셋 임포트 설정
- **FBX** `Games/Library/Models/MultiStoryBookWall.fbx` 선택 → Inspector
  - `Model` 탭: 크기가 너무 크/작으면 **Scale Factor** 조정
  - `Model` 탭: **Generate Colliders 체크** (플레이어가 벽·책장에 안 뚫고 다니게)
  - `Materials` 탭: `Extract Materials...` (질감 안 보이면)
- **책장 이미지** `Games/Library/Sprites/bookshelf.png` 선택 → Inspector
  - **Texture Type = `Default`** 로! (책 크롭 기능이 RawImage를 쓰기 때문. Sprite 아님)
  - `Non-Power of 2 = None`, `Wrap Mode = Clamp`

## 1. 씬 만들기
- `Games/Library/Scenes/` 에 새 씬 (예: `LibraryExplore`)
- FBX를 씬으로 드래그, 위치 (0,0,0)
- 바닥에 콜라이더 없으면 → `3D Object > Plane` 추가(콜라이더 포함)해서 바닥 깔기
- `Directional Light` 하나 (URP 기본)

## 2. 플레이어
빈 오브젝트 `Player` 생성:
1. `Character Controller` 추가 (Center/Radius/Height를 사람 크기로)
2. `LibraryExplorer` 스크립트 추가 (Cam에 Main Camera 연결, 비우면 자동)
3. `LibraryInteract` 스크립트 추가
4. **상호작용 감지용**: 자식 빈오브젝트에 `Sphere Collider`(Is Trigger ✅, 반경 1.5~2) + `Rigidbody`(Is Kinematic ✅, Use Gravity ❌)
   - 또는 Player 본체에 위 두 개를 추가해도 됨
5. (선택) 캐릭터 모델/캡슐을 자식으로 붙여 보이게

## 3. 카메라 (45도 뷰)
- `Main Camera` 에 `IsoCameraFollow` 추가
  - Target = Player
  - Pitch = 45, Offset = (0, 12, -12) 정도 (멀게/가깝게 조정)

## 4. 상호작용 지점
- 책장(또는 도서관 모델 전체)에 빈 오브젝트 + `Box Collider`(Is Trigger ✅, 영역 크게) + `InteractZone` 스크립트
- 여러 책장에 각각 두거나, 도서관 입구에 하나만 둬도 됨

## 5. 책 찾기 UI (Canvas)
`GameObject > UI > Canvas` (Screen Space - Overlay) + `EventSystem`(자동)
> ⚠️ EventSystem은 **InputSystemUIInputModule** 이어야 클릭됨 (신 Input System). 자동으로 안 되면 교체.

Canvas 안에 **FindScreen 패널** 구성:
- `Panel` (전체 화면) → `FindBookScreen.panel`
  - `RawImage` (ShelfImage) → bookshelf.png 의 **Texture** 연결, 화면 크게. → `shelfImage`
  - 빈 `RectTransform` (Buttons) → **ShelfImage와 똑같은 위치·크기로 덮기**(stretch). → `buttonsParent`
  - `RawImage` (TargetThumb) 작게(예: 우측 상단) → `targetThumb` (찾을 책이 여기 크롭돼서 뜸)
  - `TMP Text` (Info) → `infoText`
  - `ClearPanel`, `FailPanel` (각각 패널, 평소 꺼둠) → `clearPanel`, `failPanel`
- Panel에 `FindBookScreen` 스크립트 추가 → 위 항목들 전부 연결

그리고 **Player의 LibraryInteract** 에:
- `findScreen` = FindBookScreen
- `prompt` = "[E] 책 찾기" 안내 UI (작게 하나)
- `explorer` = Player의 LibraryExplorer

## 6. 책 격자 맞추기 (중요)
`FindBookScreen` 의 **Cabinets** 리스트가 이미지의 책 위치를 격자로 잡아요. 기본값은 2개(좌/우 책장, 각 7열 3행)예요.
- 책장 이미지에 안 맞으면 각 Cabinet의 **uv (x,y,width,height: 0~1)** 와 cols/rows 를 조정
- ▶ Play 해서 목표 책 크롭과 실제 클릭 위치가 맞는지 보며 미세조정

## 7. 난이도 조절 (FindBookScreen 인스펙터)
- `Books To Find`: 연속으로 찾을 권 수 (기본 5)
- `Time Limit`: 제한시간 초 (기본 30)
- `Max Wrong`: 허용 오답 (기본 3)
- → 비슷한 책이 많아 기본적으로 어렵고, 위 값으로 더 빡세게 가능

## ▶ 플레이
WASD로 도서관 구경 → 책장 근처서 **E** → 우상단 "찾을 책" 보고 책장에서 클릭 → 시간 내 N권!
